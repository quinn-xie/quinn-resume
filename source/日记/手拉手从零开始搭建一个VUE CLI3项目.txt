
1.安装NODE 
http://nodejs.cn/download/ 下载相应的安装程序进行安装

2.安装NPM，一般node会自带上npm

3.Windows系统 【WIN+R】运行cmd命令
node Cv / npm Cv 打印出版本号即表示安装成功

4.安装VUE CLI3 依赖
npm install -g @vue/cli-service-global  安装全局依赖

5.创建项目 my-project
到相应目录：eg / cd share.xie\test
方法一： vue create my-project
方法二： vue ui 图形化介面设置
创建完成后运行npm run serve 成功

6.集成sass/scss到vue项目
方式一：在安装vue cli3的时候选择手动，在步骤中选择添加
方式二：npm install --save node-sass --registry=https://registry.npm.taobao.org --disturl=https://npm.taobao.org/dist --sass-binary-site=http://npm.taobao.org/mirrors/node-sass
说明：
--registry=https://registry.npm.taobao.org 淘宝npm包镜像
--disturl=https://npm.taobao.org/dist 淘宝node源码镜像，一些二进制包编译时用
--sass-binary-site=http://npm.taobao.org/mirrors/node-sass 这个才是node-sass镜像

第一天，安装脚手架基本走通
vue cli3与cli2有很大不同，官方称要做到零配置，所以少了config和build文件夹，也就少了vue.config.js文件，所以在需要配置的时候就有些懵B了。
今天走通了基本的路由动态切换，SASS在项目中的使用，以及keep-alive的使用。
